# Python Short Practices

A collection of small, self-contained **Python scripts and practice projects** designed to reinforce fundamental programming skills and concepts.

---

## Projects Included

This repository currently features the following mini-projects:

* **`todolist.py`**: A simple command-line To-Do List application that allows users to add, view, and remove tasks.
* **`fortune.py`**: A classic Magic 8-Ball implementation that provides random, predetermined responses to user questions.

---

## Prerequisites

To run any of the scripts in this repository, you only need:

* **Python 3.x** (installed on your system)

---

## Getting Started

### Installation

1.  **Clone the repository** to your local machine:

    ```bash
    git clone [https://github.com/rupkatha-tech/Python-short-practices.git](https://github.com/rupkatha-tech/Python-short-practices.git)
    ```

2.  **Navigate** to the project directory:

    ```bash
    cd Python-short-practices
    ```

### Usage

To run any of the practice scripts, execute them directly using the `python` command:

**To run the To-Do List:**
```bash
python todolist.py

python fortune.py
